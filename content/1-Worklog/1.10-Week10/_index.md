### Week 10 Objectives:

* Learn and deploy AWS data analytics solutions through **Data Lake**, **AWS Glue**, **Amazon Athena**, and **Amazon QuickSight**.
* Master advanced access control with **IAM Role**, **IAM Conditions**, and IP/time-based access restrictions.
* Practice building a centralized analytics system that supports cloud-based data processing and visualization.
* Study **Generative AI on AWS** by integrating **Amazon DynamoDB**, **Amazon OpenSearch Service**, and **Amazon Bedrock**.
* Improve security administration, data monitoring, and enterprise data analytics workflows on AWS.

### Results achieved in week 10:

#### Data Analytics and Data Lake:

* Understood **Data Lake** architecture on AWS and the role of Amazon S3 in centralized data storage.
* Practiced successfully with **AWS Glue** and understood ETL workflows and automated metadata management in big data systems.
* Used **Amazon Athena** to query data directly on Amazon S3 without building a separate database system.
* Learned and practiced data visualization with **Amazon QuickSight**, supporting reports and dashboards for decision making.
* Understood the benefits of serverless analytics for cost optimization and system scalability.

#### Access Management and Advanced Security:

* Successfully configured **IAM User**, **IAM Group**, **IAM Role**, and **Switch Role** for flexible AWS access management.
* Applied **IAM Conditions** based on IP address and login time to strengthen system security.
* Understood how to implement **Least Privilege** to reduce unauthorized access risk in enterprise environments.
* Learned dynamic authorization through IAM Role and fine-grained access control through policy conditions.

#### AI Applications in Data Analytics:

* Studied the integration architecture between **Amazon DynamoDB**, **Amazon OpenSearch Service**, and **Amazon Bedrock** in a modern Generative AI system.
* Understood **DynamoDB Zero-ETL Integration**, which synchronizes data to OpenSearch without building a traditional ETL process.
* Learned the principles of **Vector Embedding** and **Semantic Search** for meaning-based data search.
* Learned the role of **Amazon Bedrock** in building Generative AI applications on AWS.
* Evaluated the potential of AI for data analysis, intelligent search, and decision support in enterprises.

#### Cloud Data System Optimization:

* Understood how to manage and optimize data in AWS Cloud through modern data analytics services.
* Learned how to combine Data Lake, Glue, Athena, QuickSight, and AI services to build a centralized data platform.
* Evaluated scalability, processing performance, and high availability of enterprise data systems on AWS.
* Improved skills in deploying, administering, and securing cloud-based data systems.
